# healthcare-operations-dashboard
Interactive Healthcare Operations &amp; Revenue Dashboard built in Tableau to analyze hospital performance, patient workload, and financial insights.
# Objective
The goal of this project is to transform healthcare data into meaningful insights that support operational decision-making and hospital performance analysis.
# Key Metrics Analyzed
Total Patients, Admissions, and Appointments
Average Wait Time & Length of Stay
Department-wise Admissions and Appointments
ER Visit Trends and Doctor Workload
Department Revenue and Treatment Costs
Lab Test Utilization
# Key Insights
Radiology and Emergency departments handle the highest number of admissions.
Emergency visits have a higher probability of hospital admission.
Departments with higher patient volumes generate the largest share of hospital revenue.
MRI, Ultrasound, and CT Scan are the most frequently performed diagnostic tests.
